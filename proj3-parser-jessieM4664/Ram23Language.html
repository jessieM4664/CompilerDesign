Ram23 Language

The Ram23 Language is based on MiniJava, with the following distinctions:

  1. the && logical operator is represented by the symbol and
  2. or is also defined
  3. the equality relational operator is defined, but is represented by symbol eq
  4. ne for not equal; ge for greater than or equal to; le for less than or equal to; < for less than; > for greater than
  5. the addition and subtraction combination operators += and -= are defined
  6. System.out.println has been revised to simply println
  7. a print variant is also in the language, having the same semantics as the Java version
  8. the type boolean is shortened to bool
  9. the type int is lengthened to integer
  10. semantically, any int value can also be assigned to a bool variable. A value of 1 will represent true, all other values will represent false.
  11. we'll also add the traditional for loop into the language.
  12. for multiline comments, no nesting will be tested
